# Loading Backend

Backend API для проекта Loading Styling.

## Установка

```bash
yarn install
```

## Запуск

```bash
yarn start
```

Сервер запустится на порту 3000 (или на порту, указанном в переменной окружения `PORT`).

## API

### GET /api/data

Возвращает список новостей с задержкой 2 секунды (для демонстрации lazy loading).

**Ответ:**
```json
{
  "status": "ok",
  "data": {
    "title": "Новости мира кино",
    "items": [
      {
        "id": 1,
        "title": "Премьера нового блокбастера",
        "description": "В прокат вышел долгожданный фильм от известного режиссера."
      },
      ...
    ],
    "timestamp": 1234567890
  }
}
```
